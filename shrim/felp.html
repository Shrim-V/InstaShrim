Général:

  • whatshotPourquoi ce site est tellement génial?
    Il a été créé par un génie génial ... et Shrimad Vora.
  • settings_input_antennaPourquoi ce site n'est-il accessible que si j'ai le code?
    Eh bien, tout ce site est construit sur un serveur local appelé Apache. Cela signifie que pour avoir accès au site, il faut avoir le code SQL et le code php et html pour l'exécuter. C'est pourquoi nous gardons notre code sur Github, simplement en cas où vous souhaitez accéder à notre site sur la route ou à la maison.
  • contact_phoneComment puis-je vous rejoindre?
    Si vous souhaitez contacter l'un de nous, cliquez sur "plus" sur la barre de navigation et sélectionnez la page à propos de. Ici, vous trouverez toutes les informations dont vous avez besoin!

Utilisation:

  • searchComment puis-je rechercher quelque chose?
    Si vous accédez à notre page principale, il y a une barre qui indique clairement la recherche. Avec cette barre, vous pouvez rechercher un tweet ou un hashtag. Pour accéder à la page principale, cliquez sur notre magnifique logo sur la barre de navigation dans le coin supérieur gauche de la page.
  • skip_previousComment puis-je accéder à la page principale?
    C'est facile! Cliquez simplement sur notre magnifique logo sur la barre de navigation dans le coin supérieur gauche de la page.
  • languageCe site est-il disponible dans d'autres langues?
    Oui! Cependant, comme nous sommes encore un projet en développement, nous n'avons pas une multitude de langues disponibles pour nos utilisateurs, mais nous avons rendu toutes les pages plus accessibles à l'échelle mondiale en les rendant disponibles en anglais, espagnol et français.
  • languageComment puis-je changer les langues?
    Pour modifier la langue de votre page, cliquez simplement sur le drapeau du pays à qui vous souhaitez que la page se trouve. Encore une fois, il n'y a que trois langues disponibles pour les utilisateurs en ce moment.
  • library_addComment tweeter un message?
    Bonne question! Pour tweeter, tapez quelque chose dans la barre à côté du mot "tweet:".
  • library_addComment puis-je mettre des hashtags sur les tweets?
    Lorsque vous tweeter quelque chose, tapez quelque chose dans la barre à côté de "hashtag:", et ce hashtag sera associé à votre tweet.
  • note_addPuis-je mettre plus d'un hashtag sur un tweet?
    Pour sûr! Assurez-vous simplement que vous séparez chaque mot séparé avec un espace, et notre système le ramènera comme des hashtags séparés.

Securité:

  • verified_userEst-ce que d'autres utilisateurs peuvent choisir le même nom d'utilisateur que moi?
    Absolument pas. Les noms d'utilisateur sont uniques pour tout le monde et nous nous assurons que si d'autres utilisateurs ne sont pas autorisés à choisir les noms d'utilisateur déjà pris.
  • visibilityEst-ce que d'autres personnes peuvent voir ce que je tweet?
    Oui, alors assurez-vous que les messages offensifs ou blessants ne sont pas envoyés via des tweets. Nous sommes ici à Shrimstagram ne tolère aucun acte de cyberintimidation et/ou de discrimination.
  • lockEst-ce que mon compte est sécurisé?
    100%. Lorsque vous configurez un mot de passe, notre système le crypte afin que, bien que les personnes puissent avoir le même mot de passe, personne ne pourra décrypter et déterminera désormais votre mot de passe. Tu es en sécurité!
